# Backend

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start
```

## Test

```bash
# unit tests
$ npm run test
```

## API Documentation

```bash
POST /register
POST /auth
GET /refresh
GET /logout
GET /products
POST /products
POST /products/:id/reviews
GET /products/top
GET /products/:id
DELETE /products/:id      
PUT /products/:id
```
